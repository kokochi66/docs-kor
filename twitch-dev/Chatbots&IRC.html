챗봇 & IRC 시작하기 원본 사이트

개요

트위치에서는 채팅 기능을 위해서, Internet Relay Chat(IRC)라는 인터페이스를 제공합니다. 챗봇은 IRC 표준을 이용하여, 트위치의 채팅 서버와 상호작용이 가능합니다. 챗봇은 트위치 IRC 네트워크와 클라이언트 간에 상호작용을 위해 연결되어있습니다. 이 가이드는 챗봇과 IRC를 사용하기 위한 다음 단계의 이전에, 여러분들에게 챗봇 시스템을 시작하는 예제를 제공해드립니다.

봇 구성하기

우리는 누군가가 !dice라고 입력하면, 이에 맞는 응답을 돌려주는 간단한 챗봇을 만들 것입니다. 먼저 우리는 1과 6 사이의 랜덤으로 숫자를 반환해주는 트리거가 필요합니다. 우리는 Glitch에서 완벽하게 작동하는 챗봇을 만들 것이고, 이것을 여러분의 컴퓨터에서 로컬로 실행하는 방법을 보여드릴 것입니다. 결과로 나타나는 화면은 다음과 같습니다.

환경변수 가져오기

시작하기 위해서 여러분은 세가지의 환경변수가 필요합니다.

변수 설명
BOT_USERNAME

챗봇이 채팅 메세지로 표시해주는 계정의 이름(username)입니다. 이것은 보통 여러분의 트위치 계정이 될 것입니다. 대부분의 개발자들은 두 개 이상의 트위치 계정을 그들의 챗봇에 사용하기 위해서 만들기 때문에, 이 값을 이용해서 계정의 주인을 확인하기는 어렵습니다.
CHANNEL_NAME

여러분이 봇을 사용하기를 원하는 트위치의 채널의 이름입니다. 보통 이 값은 여러분의 메인 트위치 계정이 됩니다.
OAUTH_TOKEN

여러분의 챗봇을 트위치 서버에서 인증하기 위한 토큰입니다. 이것은 챗봇 계정이 로그인되어 있는 동안에 https://twitchapps.com/tmi/에서 생성할 수 있습니다. 토큰은 영어 문자열로 이루어져 있습니다.

챗봇을 Glitch에서 구동시키기

Glitch는 클라우드상에서 여러분에게 생성, 편집, 호스팅 기능을 지원해줍니다. 이러한 기능에 대한 안내 코드는 이곳에서 확인할 수 있습니다. 위의 환경변수 값을 .env파일에 입력하고, 실행하기를 클릭하면 됩니다.

클릭하면, 챗봇은 트위치 IRC 네트워크에 연결되어 실행될 것입니다. 우리는 이 봇과 twitch.tv/<CHANNEL_NAME>에서 !dice 명령어를 입력함으로써, 상호작용 할 수 있을 것입니다. 여러분들은 1-6 사이의 숫자를 확인할 수 있습니다.

챗봇을 로컬적으로 실행하기

여러분이 로컬 컴퓨터 상에서 여러분의 챗봇을 생성하여 실행해보고 싶다면, 다음 단계를 따라하세요

  1. node.js를 설치합니다.
  2. 커맨드 상에서, Node Package Manager를 이용해서 tmi.js를 설치합니다. npm install tmi.js
  3. 아래 bot.js 코드에서 세가지 변수값을 여러분들이 얻은 값으로 변경해줍니다.
  4. bot.js 를 여러분의 서버에서 로컬적으로 실행하세요. node bot.js
  5. 그러면 해당 채팅봇은 트위치 IRC 네트워크와 연결되어 실행 될 것입니다. 여러분은 twitch.tv/<CHANNEL_NAME> 에서 !dice 명령어를 입력함으로써, 상호작용 할 수 있을 것입니다. 여러분들은 1-6 사이의 숫자를 확인할 수 있습니다.

const tmi = require('tmi.js');
 
// Define configuration options
const opts = {
  identity: {
    username: <BOT_USERNAME>,
    password: <OAUTH_TOKEN>
  },
  channels: [
    <CHANNEL_NAME>
  ]
};
 
// Create a client with our options
const client = new tmi.client(opts);
 
// Register our event handlers (defined below)
client.on('message', onMessageHandler);
client.on('connected', onConnectedHandler);
 
// Connect to Twitch:
client.connect();
 
// Called every time a message comes in
function onMessageHandler (target, context, msg, self) {
  if (self) { return; } // Ignore messages from the bot
 
  // Remove whitespace from chat message
  const commandName = msg.trim();
 
  // If the command is known, let's execute it
  if (commandName === '!dice') {
    const num = rollDice();
    client.say(target, `You rolled a ${num}`);
    console.log(`* Executed ${commandName} command`);
  } else  {
    console.log(`* Unknown command ${commandName}`);
  }
}
 
// Function called when the "dice" command is issued
function rollDice () {
  const sides = 6;
  return Math.floor(Math.random() * sides) + 1;
}
 
// Called every time the bot connects to Twitch chat
function onConnectedHandler (addr, port) {
  console.log(`* Connected to ${addr}:${port}`);
}
